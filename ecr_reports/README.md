# بلاغات تطبيق ECR (الجوال)

## المسارات (API)

المسار الأساسي:

- `GET /api/ecr/catalog/conditions/` كتالوج تفاصيل الحالة المرضية
- `GET /api/ecr/catalog/services/` كتالوج الخدمات المقدمة
- `POST /api/ecr/mobile-reports/` إنشاء بلاغ من التطبيق
- `GET /api/ecr/mobile-reports/` عرض بلاغات المستخدم (المالك فقط)
- `GET /api/ecr/mobile-reports/{id}/` تفاصيل بلاغ
- `GET /api/ecr/mobile-reports/{id}/documentation/` نص جاهز لمشاركة توثيق البلاغ (للإبلاغ لـ 997)

## قيود المنطقة (اختيار الموقع داخل المنطقة)

التحقق يتم على السيرفر (وليس فقط واجهة التطبيق):

1) النظام يأخذ `region` من المستخدم `request.user.region` ويمنع إرسال أي Region ID من العميل.
2) إذا كانت منطقة المستخدم تحتوي `boundary_geojson` (Polygon) سيتم التحقق أن الإحداثيات داخل الحدود.
3) في حال كانت خارج الحدود يرجع API الرسالة: **يرجى اختيار الموقع داخل منطقتك فقط.**

> لتفعيل الحدود: أدخل GeoJSON للمنطقة من لوحة الإدارة (Regions).
